# 一些常用的全局变量
## __filename
```__filename``` 表示当前执行脚本的文件名(绝对路径)。
## __dirname
```__dirname``` 表示当前执行脚本所在的目录（绝对路径）。
## setTimeout 和 clearTimeout
定时器函数，使用方法和浏览器端一致。
## setInterval 和 clearInterval
定时器函数，使用方法和浏览器端一致。
## console 相关方法
用来提供控制台相关的操作 API，使用方法和浏览器端一致。
## process
```process``` 用来描述当前 Node 运行的进程状态。g该对象提供了一些 Node 程序和系统交互的 API。
